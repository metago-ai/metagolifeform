# 势能编织法

## 触发条件
当用户提到以下关键词或场景时触发本技能：
- "势能"、"编织"、"动能转化"、"势能放大"
- 战略规划中的资源整合
- 生态构建与能量流动分析
- 组织能量管理与动能激发

## 适用场景
- 战略规划与路径设计
- 资源整合与配置优化
- 生态构建与价值网络设计
- 组织能量管理与动能激发
- 变革推动与势能蓄积

## 执行步骤

1. **识别五种核心势能**：
   - 认知势能：知识储备、洞察力、战略思维、学习能力
   - 关系势能：人脉网络、合作伙伴、信任资本、社会资本
   - 技术势能：技术积累、专利资产、工具能力、平台基础
   - 制度势能：流程规范、激励机制、治理结构、组织设计
   - 文化势能：价值观、使命感、组织氛围、共同信念

2. **评估各势能储量**（1-10分）：
   - 储量评估：当前已积累的势能规模
   - 流动性评估：势能可被调用的难易程度
   - 转化效率评估：势能转化为动能的效率
   - 衰减率评估：势能随时间衰减的速度

3. **六步编织法**：
   - 势能识别：盘点现有势能类型与储量，绘制势能地图
   - 势能蓄积：通过持续投入积累势能，建立蓄能池
   - 势能引导：设计势能流动方向与路径，构建引导机制
   - 势能转化：将势能转化为实际动能，产生价值产出
   - 势能放大：通过杠杆效应放大势能效果，实现非线性增长
   - 势能更新：循环更新保持势能活力，避免势能耗竭

4. **选择战术工具箱中的工具**：
   - 蓄势工具：长期投资、能力建设、人才储备
   - 引势工具：战略引导、方向设定、目标对齐
   - 转势工具：项目化运作、资源调配、机制激活
   - 放势工具：杠杆利用、网络效应、生态协同
   - 更势工具：复盘迭代、持续学习、知识沉淀

5. **输出势能编织方案**

## 核心理论
- **势能编织法**：组织如同一张势能之网，五种势能相互交织、相互放大，形成非线性增长
- **势能转化定律**：势能必须通过有序引导才能转化为有效动能，无序的势能只会耗散
- **六步编织循环**：识别→蓄积→引导→转化→放大→更新，形成势能持续循环
