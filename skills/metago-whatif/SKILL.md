# 反事实推演（metago-whatif）

## 描述
推演"如果...会怎样"的假设情景。识别当前状态的关键变量，设定假设条件，推演不同选择导致的不同结果，量化结果差异，给出最优建议。

## 触发条件
用户请求反事实推演时自动激活。方案评估时自动激活（提供替代方案分析）。

## 核心流程
1. 识别当前状态的关键变量
2. 设定假设条件（"如果X不同，会怎样"）
3. 推演不同选择导致的不同结果
4. 量化结果差异
5. 给出最优建议

## 输出格式
返回结构化JSON，包含：
- 假设条件列表
- 各假设下的推演结果
- 结果差异量化
- 最优建议

## 根源文档
`元构全息智能引擎.txt`（metago-whatif技能定义(217-220, 353-370）、元进化五阶段(140-149)）

## 与其他技能的协同
- 与 `metago-critique` 协同：推演结果接受批判性分析
- 与 `metago-decision-eval` 协同：推演结果用于决策评估
