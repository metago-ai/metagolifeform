# 行动计划生成

## 描述
将目标分解为可执行的步骤序列。每个步骤必须是可执行的，有明确的依赖关系和预期耗时，高风险步骤标注提醒和回滚方案。

## 触发条件
用户请求批判分析 / 反事实推演 / 情绪检测 / 客观度评估 / 行动计划 / 决策评估时自动激活。

## 核心流程
1. 接收输入内容
2. 按对应维度进行分析评估
3. 输出结构化JSON结果
4. 附评估依据和改进建议

## 输出格式
返回结构化JSON，包含评分、各维度得分、判定结论和改进建议。
