# 28维价值评估

## 触发条件
当用户提到以下关键词或场景时触发本技能：
- "价值评估"、"多维价值"、"价值雷达"
- 投资决策与项目评估
- 战略选择与方案对比
- 价值短板识别

## 适用场景
- 投资决策与项目筛选
- 组织价值评估与诊断
- 战略方案选择与对比
- 产品价值评估
- 生态价值衡量
- 并购标的价值评估

## 执行步骤

1. **28维价值扫描**（D01-D28）：
   - D01-D07 技术创新维度：技术先进性、创新性、可扩展性、可靠性、安全性、兼容性、前瞻性
   - D08-D14 制度协同维度：制度合理性、协同效率、治理结构、激励机制、合规性、适应性、透明度
   - D15-D21 运营服务维度：运营效率、服务质量、用户体验、响应速度、成本控制、可持续性、风险控制
   - D22-D28 生态治理维度：生态贡献、合作共赢、社会责任、环境影响、文化价值、长期价值、综合效益

2. **每维评分（0-10）和权重计算**：
   - 根据评估对象确定各维度权重（W_i，0-1之间，总和为1）
   - 对每个维度进行0-10分评分（V_i）
   - 记录评分依据与证据，确保可追溯

3. **综合价值指数计算**：
   - 公式：V_total = Σ(W_i × V_i) for i = 1 to 28
   - 计算各维度群组得分（技术创新群、制度协同群、运营服务群、生态治理群）
   - 识别价值短板（得分<6的维度）和价值优势（得分≥8的维度）

4. **价值雷达图生成**：
   - 28维雷达图可视化
   - 群组得分对比
   - 价值均衡度分析（标准差计算）

5. **价值短板识别+提升建议**

## 核心理论
- **28维价值共振体系**：价值是多维共振的整体表现，单一维度的高分不能代表整体价值，需多维协同
- **加权综合评估**：V_total = Σ(W_i × V_i)，权重反映各维度的相对重要性
- **价值均衡原理**：高价值系统要求各维度均衡发展，避免短板效应拖累整体价值
