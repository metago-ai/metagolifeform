# 反事实推演

## 描述
推演"如果...会怎样"的假设情景。识别当前状态的关键变量，设定假设条件，推演不同选择导致的不同结果，量化结果差异，给出最优建议。

## 触发条件
用户请求批判分析 / 反事实推演 / 情绪检测 / 客观度评估 / 行动计划 / 决策评估时自动激活。

## 核心流程
1. 接收输入内容
2. 按对应维度进行分析评估
3. 输出结构化JSON结果
4. 附评估依据和改进建议

## 输出格式
返回结构化JSON，包含评分、各维度得分、判定结论和改进建议。
