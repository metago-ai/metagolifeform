# 三元五纬诊断模型

## 触发条件
当用户提到以下关键词或场景时触发本技能：
- "组织诊断"、"健康度评估"、"三元失耦"、"五纬扫描"
- 企业变革前的现状评估
- 团队效能与组织健康度分析
- 数字化转型成熟度评估
- 重组并购后的整合诊断

## 适用场景
- 企业组织健康度诊断
- 团队效能与协作评估
- 组织变革前的基础诊断
- 数字化转型成熟度评估
- 重组并购后的整合诊断

## 执行步骤

1. **三元本体分析**：分析碳基（人）、硅基（系统）、比特（数据）三元本体的深度耦合度
   - 碳基（人）：组织成员的数字素养、协作能力、决策权限
   - 硅基（系统）：技术架构、流程系统、工具链的连通性
   - 比特（数据）：数据资产、信息流、知识图谱的完整度

2. **五纬雷达扫描**：从五个维度扫描组织状态
   - 共识度：组织成员对战略目标、价值观的认同程度
   - 协同度：跨部门、跨层级协作效率与顺畅度
   - 创新度：组织学习能力与持续创新动能
   - 韧制度：组织应对变化与冲击的恢复能力
   - 价值度：价值创造与交付的实际产出

3. **矩阵诊断**：四象限定位组织类型
   - 引领者（高三元耦合+高五纬得分）
   - 空心节点（高三元耦合+低五纬得分）
   - 孤胆天才（低三元耦合+高五纬得分）
   - 停滞单元（低三元耦合+低五纬得分）

4. **失耦诊断**：识别三元割裂的具体表现
   - 碳基与比特割裂：人员与数据脱节，决策缺乏数据支撑
   - 硅基与碳基割裂：系统与人脱节，工具未能赋能于人
   - 比特与硅基割裂：数据与系统脱节，数据孤岛严重

5. **输出诊断报告**：包含诊断结论与干预建议

## 核心理论
- **三元本体论**：碳基（人）、硅基（系统）、比特（数据）三元的有机统一，任何一元缺失或失衡都会导致组织失耦
- **耦合公式**：C_total = C_carbon × C_silicon × C_bit（任一元为0则总耦合度为0，体现木桶效应）
- **熵增公式**：组织无序度持续增加，需通过负熵注入维持有序，三元耦合是负熵的核心来源
