# 决策评估

## 描述
评估决策质量和潜在风险。评估维度：逻辑完整性、证据充分性、风险可控性、替代方案、可执行性。返回0-100分及各维度评分。

## 触发条件
用户请求批判分析 / 反事实推演 / 情绪检测 / 客观度评估 / 行动计划 / 决策评估时自动激活。

## 核心流程
1. 接收输入内容
2. 按对应维度进行分析评估
3. 输出结构化JSON结果
4. 附评估依据和改进建议

## 输出格式
返回结构化JSON，包含评分、各维度得分、判定结论和改进建议。
