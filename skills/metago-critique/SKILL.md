# 批判性分析（metago-critique）

## 描述
对任何输入进行L1-L5分级批判性分析，检测逻辑漏洞、认知偏差、事实错误、论证不充分、概念混淆。基于D45批判分级原则和D38绝对客观中立原则。

## 触发条件
用户请求批判分析时自动激活。重大决策输出前强制触发（与决策锁协同）。

## 核心流程
1. 接收输入内容
2. 按L1-L5分级进行批判性分析：
   - L1：逻辑漏洞检测
   - L2：认知偏差检测
   - L3：事实错误检测
   - L4：论证充分性检测
   - L5：概念混淆检测
3. 输出结构化JSON结果
4. 附评估依据和改进建议

## 输出格式
返回结构化JSON，包含：
- 总评分（0-100）
- 各维度得分（L1-L5）
- 判定结论
- 改进建议

## 根源文档
`元构全息智能引擎.txt`（D38绝对客观中立原则(80)、D45批判分级原则(87)、metago-critique技能定义(199-289)）

## 与其他技能的协同
- 与 `metago-decision-lock` 协同：决策锁通过后触发批判性分析
- 与 `metago-objectivity` 协同：批判分析包含客观性维度
- 与 `metago-fact-check` 协同：L3事实错误检测联动事实核查
