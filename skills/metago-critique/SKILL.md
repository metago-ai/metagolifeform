# 批判性分析

## 描述
对任何输入进行L1-L5分级批判性分析，检测逻辑漏洞、认知偏差、事实错误、论证不充分、概念混淆。基于D45批判分级原则和D38绝对客观中立原则。

## 触发条件
用户请求批判分析 / 反事实推演 / 情绪检测 / 客观度评估 / 行动计划 / 决策评估时自动激活。

## 核心流程
1. 接收输入内容
2. 按对应维度进行分析评估
3. 输出结构化JSON结果
4. 附评估依据和改进建议

## 输出格式
返回结构化JSON，包含评分、各维度得分、判定结论和改进建议。
