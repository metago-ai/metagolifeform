## 执行流程

### 步骤1：多维度评分
对当前决策或输出在29个维度上逐一评分（0-1）。

### 步骤2：加权求和
综合价值指数 = Σ(维度得分 × 维度权重)，总和=1

### 步骤3：冲突检测与解决
当任意两维冲突时（一维需要牺牲另一维）：
- 寻找互补解而非折衷
- 互补解条件: dim_i ≥ dim_i0 + dim_j0 - dim_j

### 步骤4：伦理风险计算
R = max(R₁, R₂, ..., R₁₃)
- R ≥ 0.6: 🔴 红灯阻断
- 0.3 ≤ R < 0.6: 🟡 黄灯警示
- R < 0.3: 🟢 通过

### 步骤5：权重自适应
每日基于政策密度、用户需求、耦生度变化等因素自动更新权重。
